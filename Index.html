Markup is code that tells the browser how the text on a web page should look like.

Block elements have invible boxes which have some features:

  1. margin
  2. height
  3. width
  4. border

Types of block elements

  • p - paragraph
  • ol, ul - lists
  • headlines
    1. h1
    2. h2
    3. h3
    4. h4
    5. h5
    6. h6
  • video - video player
  • div - generic block
  • table - table element

li element and p element have optional closing tags.

A browser knows when to close an element with optional closing tag when the same element is present again with an opening tag, so the browser knows that the new piece of element is mentioned because the first one must have ended since those elements can't be inside one another.
